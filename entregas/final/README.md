# Proyecto Final Backend (CoderHouse 2022)

## Rutas
---------------
### Login/Acceder a la API
```
/shop
```

### GET
```
/api/products
/api/products/:id
/api/products/category/:category
/api/mensajes
/api/mensajes/:alias
/api/msgsto
/api/msgsto/:alias
/api/orders
/api/cart
/api/cart/:id/products
```

### POST
```
/api/products
/api/mensajes
/api/msgsto
/api/cart
/api/cart/:id/products
/api/orders
```

### PUT
```
/api/products/:id
```

### DELETE
```
/api/products/:id
/api/cart/:id
/api/cart/:id/products/:id_prod
```

## Comandos
---------------
### Iniciar API
```
npm start
```

### Sin JWT
```
npm run nojwt
```
